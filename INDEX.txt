## âš ï¸ Important Notes

### Refresh Schedules
- Only **enabled** schedules are restored
- **Disabled** schedules are skipped
- This prevents API validation errors

### Security
- Keep `.env` file secure
- Don't commit `.env` to version control
- Rotate secrets regularly
- Use HTTPS in production

### Performance
- Small workspace: 2-5 minutes
- Medium workspace: 5-15 minutes
- Large workspace: 15+ minutes
